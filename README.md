# 介绍
一款由我学习 Unity 过程中开发的一款2D休闲手游。其中游戏框架的设计，部分界面的绘制，动画，逻辑实现等都由个人完成。

## 游戏特点
+ 操作简单易上手
+ 支持中英语言切换
+ 其中引入了简单的商店系统
+ 配置界面可配置BGM，音量大小，以及一些特效（还不够完善）
+ 有一个比较完整的关卡流程
+ 有较为简易的结算系统和界面
+ 有教程关卡
